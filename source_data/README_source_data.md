# Source data

This directory contains processed source data, figure-source files, and supplementary table-source files used in the manuscript and Supplementary Information. The complete production dataset can be regenerated from the scripts and configuration in the repository root.

Subdirectories:

- `tables/`: CSV/TEX source tables and processed summaries.
- `figures/`: selected main-manuscript figure-source PDFs.
- `supplementary_tables/`: supplementary table-source files, including Supplementary Table S12 and its per-variant attempt logs.
- `supplementary_figures/`: Supplementary Fig. S1-S8 source PDFs.

The production workflow generated 320000 per-step observations. This compact release includes condition-step summaries and run-level transient metrics rather than the full long-format per-run/per-step table. That long-format file can be regenerated by running the production and analysis pipeline.

Validation: run `python src/validate_submitted_source_data.py --root .` from the archive root, or use `bash scripts/06b_validate_submitted_source_data.sh`, to check the compact source-data package included in the release.
